Функция ltrim

  • Определение

    ltrim - удаляет пробелы (или другие символы) из начала строки

  • Пример

    $var = "     Доброе утро!";

echo ltrim($var); // "Доброе утро!"

  • Синтаксис

    ltrim(string[,characters])

  • Параметры

    Параметр Описания
    string Обязательный. Строка над которой будут производится действия по удалению символов слева
    characters Необязательный. С помощью параметра characters можно также указать удаляемые символы

  • Возвращаемое значение

    Тип Описания
    Строка Строка с удалёнными из начала строки пробелами
